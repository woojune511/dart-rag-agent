# v0.1.0 Runtime Contract Ready

Release date: 2026-06-08

This checkpoint freezes the current single-document DART runtime contract before
the next cleanup and hardening phase.

## Scope

- Evidence-backed numeric QA over DART filings.
- Deterministic calculation execution with structured operand, result, and trace
  artifacts.
- Critic acceptance and reflection gates exposed through reviewer-facing
  projections.
- Report-scoped cache capability evidence kept candidate-only, with serving and
  live ledger insertion disabled.
- Portfolio review gates ready for handoff.

## Operational Boundary

This release does not promote disabled capabilities. In particular:

- report-cache serving remains disabled;
- retrieval bypass remains disabled;
- live ledger insertion from report cache remains disabled;
- final acceptance authority remains with the critic/orchestrator contract.
